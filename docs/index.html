Marionette

Powered by the Puppetmaster orchestrator

Not a chat. A swarm.

Every other coding agent is one model in a chat loop. Marionette is a desktop agent built on Puppetmaster — an orchestration engine that splits work into parallel worker swarms and routes each task to the cheapest model that can do it. The LLM is a swappable part; the orchestration is the product.

  • Swarms, not a single chat. Puppetmaster fans work out to many workers running in parallel, each a real subprocess with durable, replayable, auditable state — not one linear conversation.
  • Any model, cheapest that fits. Puppetmaster's tiered router picks the model per task, so trivial work runs near-free and only the hard parts hit a frontier model — you are never locked to one pricey LLM.
  • Yours, end to end. It runs from source over a live CodeGraph of your repo, edits its own code, and self-updates — the same on macOS, Windows, and Linux.
