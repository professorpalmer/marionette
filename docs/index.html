Marionette

Marionette is a desktop AI coding harness: an Electron shell over a source-run Python backend with CodeGraph-aware retrieval and Puppetmaster worker delegation. Install clones the repo, builds a local venv with uv, and compiles native modules on your machine -- the same Hermes-style model on macOS, Linux, and Windows.

macOS / Linux

curl -fsSL https://professorpalmer.github.io/marionette/install.sh | bash

Windows (PowerShell)

irm https://professorpalmer.github.io/marionette/install.ps1 | iex
